# person-tool-web（日本語）

このプロジェクトはローカルで動作するファイル検索・閲覧ツールです。

- キーワード検索（ファイルパス / ファイル内容）
- マンガ風画像ビューア（タグ、メモ、削除、リネーム）
- 動画ビューア（ストリーミング再生、タグ、メモ、削除、リネーム）
- 注釈一覧（タグ、メモ、評価、メディア種別フィルター）
- タスク管理（タスク種類、期限、Markdown、画像貼り付け/ドラッグ、Excel 出力）

## 言語

- [中文](./README.zh-CN.md)

## 動作環境

- Node.js 18+（推奨: Node.js 20+）

## インストール

```bash
npm install
```

## 起動

```bash
npm start
```

起動後、ターミナルにローカルURLが表示されます（例）:

- `http://127.0.0.1:3847/`（キーワード検索）
- `http://127.0.0.1:3847/manga.html`（画像ビューア）
- `http://127.0.0.1:3847/video.html`（動画ビューア）
- `http://127.0.0.1:3847/overview.html`（注釈一覧）
- `http://127.0.0.1:3847/task.html`（タスク管理）

## データとログ

- タグ・メモ・評価の保存先: `data/file-tags.json`
- タスクと進捗の保存先: `data/tasks.json`
- タスク添付（画像・ファイル）の保存先: `data/task/<タスクID>/`（タスク削除時にディレクトリごと削除）
- 旧 Markdown 用のフラット保存先 `data/task-assets/` は読み取り専用で引き続き利用可能
- 実行ログ: `logs/app-YYYY-MM-DD.log`

環境変数:

- `KW_WEB_NO_LOG=1`: ファイルログを無効化
- `KW_WEB_LOG_LEVEL=info|warn|error`: ログレベル設定

## 評価

- 評価範囲: `1~10`
- `0` は未評価（UIでは「未評」表示）
- 一覧はデフォルトで評価の高い順に並びます

## Task 機能メモ

- タスクリストは「状態 / タスク種類 / キーワード」で絞り込み可能
- タスク詳細ページで Markdown 形式の進捗を追記可能
- タスク詳細/進捗入力欄に画像を貼り付け・ドラッグすると自動アップロードし、Markdown 画像リンクを挿入
- Excel 出力（`.xlsx`）は現在の絞り込み結果を出力
